## LeetCode 977: Squares of a Sorted Array

Because the input array is already sorted, the largest square will come from either:

```python
nums[left]
```

or

```python
nums[right]
```

Why?

Because negative numbers can become large after squaring.

Example:

```python
nums = [-4, -1, 0, 3, 10]
```

The biggest square is either from:

```python
-4 ** 2 = 16
```

or:

```python
10 ** 2 = 100
```

So we use **two pointers**.

## Code

```python
from typing import List

class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        n = len(nums)
        res = [0] * n

        left = 0
        right = n - 1
        pos = n - 1

        while left <= right:
            left_square = nums[left] * nums[left]
            right_square = nums[right] * nums[right]

            if left_square > right_square:
                res[pos] = left_square
                left += 1
            else:
                res[pos] = right_square
                right -= 1

            pos -= 1

        return res
```

## Explanation

We create an output array:

```python
res = [0] * len(nums)
```

Then we fill it from the back.

Why from the back?

Because the biggest square should go at the end of the sorted result.

Example:

```python
nums = [-4, -1, 0, 3, 10]
```

Start:

```python
left = 0      # -4
right = 4     # 10
pos = 4       # last index of result
```

Compare:

```python
(-4)^2 = 16
10^2 = 100
```

`100` is bigger, so put it at the end:

```python
res = [0, 0, 0, 0, 100]
```

Move `right` left:

```python
right = 3
pos = 3
```

Now compare:

```python
(-4)^2 = 16
3^2 = 9
```

`16` is bigger, so put it next:

```python
res = [0, 0, 0, 16, 100]
```

Move `left` right:

```python
left = 1
pos = 2
```

Continue until all spots are filled.

Final answer:

```python
[0, 1, 9, 16, 100]
```

## Why `left <= right`?

Because when `left == right`, there is still one number left to square and place into the result.

For example:

```python
nums = [-1]
```

We still need to process `-1`.

So we use:

```python
while left <= right:
```

## Complexity

```python
Time: O(n)
Space: O(n)
```

We visit each number once.
