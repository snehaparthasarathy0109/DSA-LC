This is the **Container With Most Water** problem.

## Code

```python
from typing import List

class Solution:
    def maxArea(self, height: List[int]) -> int:
        res = 0

        l = 0
        r = len(height) - 1

        while l < r:
            width = r - l
            current_height = min(height[l], height[r])
            area = width * current_height

            res = max(res, area)

            if height[l] > height[r]:
                r -= 1
            else:
                l += 1

        return res
```

## Idea

You are given an array where each value represents the height of a vertical line.

You want to choose **two lines** that can hold the most water.

The amount of water depends on:

```python
width = right_index - left_index
height = min(left_line_height, right_line_height)
area = width * height
```

Why `min`?

Because the water spills over the shorter wall.

For example:

```python
height = [1, 8, 6, 2, 5, 4, 8, 3, 7]
```

If we choose index `1` and index `8`:

```python
left height = 8
right height = 7
width = 8 - 1 = 7
height = min(8, 7) = 7
area = 7 * 7 = 49
```

So the answer is `49`.

## Why two pointers?

Start with the widest possible container:

```python
l = 0
r = len(height) - 1
```

Then calculate the area.

After that, move the pointer with the **smaller height**.

Why?

Because the current area is limited by the shorter wall.

For example, if:

```python
height[l] = 1
height[r] = 7
```

The container height is only `1`.

Moving the taller wall does not help, because the shorter wall is still limiting the water.

So we move the shorter wall and hope to find a taller one.

## Pointer movement

```python
if height[l] > height[r]:
    r -= 1
else:
    l += 1
```

This means:

* If the left wall is taller, move the right pointer.
* Otherwise, move the left pointer.

We always move the shorter wall.

## Walkthrough

Input:

```python
height = [1, 8, 6, 2, 5, 4, 8, 3, 7]
```

Start:

```python
l = 0
r = 8
```

Area:

```python
width = 8
height = min(1, 7) = 1
area = 8
res = 8
```

Left wall is shorter, so move `l`.

Now:

```python
l = 1
r = 8
```

Area:

```python
width = 7
height = min(8, 7) = 7
area = 49
res = 49
```

Now the best answer is `49`.

The algorithm keeps checking possible containers until the pointers meet.

## Complexity

```python
Time: O(n)
Space: O(1)
```

Why?

Each pointer moves at most `n` times total, and we only use a few variables.
