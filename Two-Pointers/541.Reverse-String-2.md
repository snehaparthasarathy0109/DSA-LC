# Reverse Words In-Place (Bug + Fix)

## ❌ Problem in Original Code

The issue is that the pointer `i` does **not move past the space** after processing a word.

After reversing a word:
- `i` is still pointing at `' '`
- The loop runs again → infinite loop / stuck behavior

---

## 🧠 Key Bug

```python
start = i + 1
````

You update `start`, but **never increment `i`**, so it stays on the space.

---

## ✅ Fixed Solution

```python
def reverseWords(arr):
    def reverse(left, right):
        while left < right:
            arr[left], arr[right] = arr[right], arr[left]
            left += 1
            right -= 1

    # Step 1: Reverse entire array
    reverse(0, len(arr) - 1)

    start = 0
    i = 0

    # Step 2: Reverse each word
    while i < len(arr):
        while i < len(arr) and arr[i] != ' ':
            i += 1

        reverse(start, i - 1)

        i += 1          # ✅ Move past space
        start = i       # Next word start

    return arr
```

---

## 📌 Example

```python
arr = list("hello world sneha")
reverseWords(arr)
print("".join(arr))
```

### Output:

```
sneha world hello
```

---

## ⚡ Summary

* Reverse entire string
* Reverse each word individually
* **Always move pointer `i` past spaces**

```python
i += 1  # critical fix
```
