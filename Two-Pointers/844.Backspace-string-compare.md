## LeetCode 844: Backspace String Compare

We need to compare what the two strings look like **after applying backspaces**.

`#` means delete the previous character, if one exists.

---

## Simple solution: use a stack

For each string:

* If the character is not `#`, add it.
* If the character is `#`, remove the previous character if possible.

Then compare the final results.

## Code

```python
class Solution:
    def backspaceCompare(self, s: str, t: str) -> bool:
        def build(string):
            stack = []

            for ch in string:
                if ch != "#":
                    stack.append(ch)
                elif stack:
                    stack.pop()

            return stack

        return build(s) == build(t)
```

---

## Example 1

```python
s = "ab#c"
```

Process it:

```python
'a'  -> ['a']
'b'  -> ['a', 'b']
'#'  -> ['a']
'c'  -> ['a', 'c']
```

Final result:

```python
"ac"
```

Now:

```python
t = "ad#c"
```

Process it:

```python
'a'  -> ['a']
'd'  -> ['a', 'd']
'#'  -> ['a']
'c'  -> ['a', 'c']
```

Final result:

```python
"ac"
```

Both are equal, so return:

```python
True
```

---

## Why `elif stack`?

```python
elif stack:
    stack.pop()
```

This handles the rule:

> Backspacing an empty text keeps it empty.

Example:

```python
s = "###a"
```

The first three `#` characters do nothing because there is nothing to delete.

Then `a` gets added.

Final result:

```python
"a"
```

So we only pop when the stack is not empty.

---

## More optimal solution: two pointers from the back

The stack solution is easier and accepted, but it uses extra space.

We can also solve it in `O(1)` extra space by scanning both strings from right to left.

## Code

```python
class Solution:
    def backspaceCompare(self, s: str, t: str) -> bool:
        i = len(s) - 1
        j = len(t) - 1

        def get_next_valid_char_index(string, index):
            skip = 0

            while index >= 0:
                if string[index] == "#":
                    skip += 1
                    index -= 1
                elif skip > 0:
                    skip -= 1
                    index -= 1
                else:
                    break

            return index

        while i >= 0 or j >= 0:
            i = get_next_valid_char_index(s, i)
            j = get_next_valid_char_index(t, j)

            if i >= 0 and j >= 0 and s[i] != t[j]:
                return False

            if (i >= 0) != (j >= 0):
                return False

            i -= 1
            j -= 1

        return True
```

---

## How the two-pointer version works

Instead of building the final strings, we find the next real visible character from the end.

Example:

```python
s = "ab#c"
```

From the back:

```python
c
```

is visible.

Before that:

```python
#
```

deletes:

```python
b
```

So the next visible character is:

```python
a
```

The final visible characters from right to left are:

```python
c, a
```

We do this for both strings and compare each visible character.

## Complexity

### Stack solution

```python
Time: O(n + m)
Space: O(n + m)
```

### Two-pointer solution

```python
Time: O(n + m)
Space: O(1)
```

For interviews, I would start with the stack solution, then mention the two-pointer optimized version.

You are seeing:

```python
IndexError: pop from empty list
```

because your code is doing:

```python
stack.pop()
```

when `stack` has no characters inside it.

For this problem, `#` means backspace, but:

> If the text is already empty, backspace does nothing.

So before popping, you must check whether the stack is non-empty.

## Wrong version

```python
def build(string):
    stack = []

    for ch in string:
        if ch != "#":
            stack.append(ch)
        else:
            stack.pop()

    return stack
```

This fails for inputs like:

```python
s = "#a"
```

When the first character is `#`, `stack` is still empty, so `stack.pop()` crashes.

## Fixed version

```python
class Solution:
    def backspaceCompare(self, s: str, t: str) -> bool:
        def build(string):
            stack = []

            for ch in string:
                if ch != "#":
                    stack.append(ch)
                elif stack:
                    stack.pop()

            return stack

        return build(s) == build(t)
```

## Why `elif stack` fixes it

```python
elif stack:
    stack.pop()
```

means:

```python
if ch is "#" and stack is not empty:
    delete the previous character
```

But if `stack` is empty, do nothing.

Example:

```python
s = "##a#"
```

Process:

```python
#    stack empty, do nothing
#    stack empty, do nothing
a    stack = ["a"]
#    pop "a", stack = []
```

Final result is empty, and no error happens.
