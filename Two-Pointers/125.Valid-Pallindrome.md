Two main issues:

## 1. You are not handling digits

For **Valid Palindrome**, alphanumeric means:

* letters
* digits

But your `isAlpha` only allows letters:

```python
def isAlpha(self, c):
    return 'a' <= c <= 'z' or 'A' <= c <= 'Z'
```

So something like `"0P"` would be handled incorrectly.

You need digits too.

---

## 2. You are not lowercasing before comparing

Palindrome check here should be **case-insensitive**.

So:

```python
if s[left] != s[right]:
```

is wrong for cases like:

```python
"A man, a plan, a canal: Panama"
```

because `'A'` and `'a'` should be treated as equal.

You should compare:

```python
if s[left].lower() != s[right].lower():
```

---

## 3. Possible index out of bounds while skipping

These lines:

```python
while not self.isAlpha(s[left]):
    left += 1
while not self.isAlpha(s[right]):
    right -= 1
```

can run past the bounds if the string has many non-alphanumeric characters.

You should guard them with `left < right`.

---

# Fixed version

```python id="5cwvyb"
class Solution:
    def isPalindrome(self, s: str) -> bool:
        left = 0
        right = len(s) - 1

        while left < right:
            while left < right and not self.isAlphaNum(s[left]):
                left += 1
            while left < right and not self.isAlphaNum(s[right]):
                right -= 1

            if s[left].lower() != s[right].lower():
                return False

            left += 1
            right -= 1

        return True

    def isAlphaNum(self, c):
        return (
            'a' <= c <= 'z' or
            'A' <= c <= 'Z' or
            '0' <= c <= '9'
        )
```

---

# Even simpler Python version

Python has a built-in for this:

```python id="0kr9be"
class Solution:
    def isPalindrome(self, s: str) -> bool:
        left = 0
        right = len(s) - 1

        while left < right:
            while left < right and not s[left].isalnum():
                left += 1
            while left < right and not s[right].isalnum():
                right -= 1

            if s[left].lower() != s[right].lower():
                return False

            left += 1
            right -= 1

        return True
```
