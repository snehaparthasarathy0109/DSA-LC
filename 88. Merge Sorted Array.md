## LeetCode 88: Merge Sorted Array

You are given two sorted arrays:

```python
nums1
nums2
```

`nums1` has extra space at the end to hold all elements from `nums2`.

You need to merge `nums2` into `nums1` **in-place**.

---

## Key idea

Because `nums1` has empty space at the end, we should fill from the **back**.

Why?

If we fill from the front, we may overwrite useful values in `nums1`.

So we use three pointers:

```python
p1 = m - 1          # last real element in nums1
p2 = n - 1          # last element in nums2
p = m + n - 1       # last position in nums1
```

At each step, put the larger value at the end.

---

## Code

```python
from typing import List

class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        """
        Do not return anything, modify nums1 in-place instead.
        """
        p1 = m - 1
        p2 = n - 1
        p = m + n - 1

        while p1 >= 0 and p2 >= 0:
            if nums1[p1] > nums2[p2]:
                nums1[p] = nums1[p1]
                p1 -= 1
            else:
                nums1[p] = nums2[p2]
                p2 -= 1

            p -= 1

        while p2 >= 0:
            nums1[p] = nums2[p2]
            p2 -= 1
            p -= 1
```

---

## Example

```python
nums1 = [1, 2, 3, 0, 0, 0]
m = 3

nums2 = [2, 5, 6]
n = 3
```

Pointers start here:

```python
p1 = 2   # nums1[2] = 3
p2 = 2   # nums2[2] = 6
p = 5    # last index of nums1
```

Compare:

```python
nums1[p1] = 3
nums2[p2] = 6
```

`6` is bigger, so put it at the end:

```python
nums1 = [1, 2, 3, 0, 0, 6]
```

Then move `p2` and `p`.

Continue comparing from the back.

Final result:

```python
[1, 2, 2, 3, 5, 6]
```

---

## Why only copy leftover `nums2`?

At the end, we do:

```python
while p2 >= 0:
    nums1[p] = nums2[p2]
    p2 -= 1
    p -= 1
```

Because if `nums2` still has elements, they need to be copied into `nums1`.

But if `nums1` still has elements, they are already in the right place.

Example:

```python
nums1 = [1, 2, 3, 0, 0, 0]
nums2 = [4, 5, 6]
```

After placing `6`, `5`, and `4`, the leftover `1, 2, 3` are already correctly placed.

---

## Complexity

```python
Time: O(m + n)
Space: O(1)
```

We modify `nums1` in-place.
